# **App Name**: Eventide Campus

## Core Features:

- Student Registration: Allows students to register for events, automatically joining the event's society and receiving a confirmation email.
- QR Code Generation and Check-in: Generates QR codes for registered students, enabling quick check-ins at events via QR scan, updating attendance records.
- Society Admin Event Creation: Enables society admins to create and manage events, subject to super-admin approval for verification.
- Automated Email Reminders: Sends automated email reminders to students about upcoming events 24 hours before they occur.
- Role-Based Access Control (RBAC): Implements RBAC to manage permissions for students, society admins, and super admins.
- Event Recommendations: AI powered tool for suggesting relevant events to students based on their profiles and preferences.
- Super Admin Analytics Dashboard: Provides a dashboard for super admins to view analytics, approve society admins, verify events, and manage users.

## Style Guidelines:

- Primary color: Deep Blue (#2E3192), reflecting trust and sophistication, resonating with the educational environment.
- Background color: Light blue (#D3E4EF), a soft and clean background for comfortable readability and a modern feel.
- Accent color: Vibrant Orange (#FF4500) used strategically for key actions and highlights to draw attention.
- Body and headline font: 'Inter', a sans-serif font, chosen for its modern, clean appearance suitable for both headlines and body text.
- Note: currently only Google Fonts are supported.
- Crisp, minimalist icons for easy navigation.
- Modern SaaS dashboard layout featuring a sidebar, top navbar, and glassmorphism cards.
- Smooth transitions and Framer Motion animations to enhance user experience.
